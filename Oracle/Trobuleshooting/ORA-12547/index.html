# ORA-12547 TNS:lost contact

# Case

操作系统：Oracle Linux Server 7.9

数据库：11.2.0.4 GI for a standalone server

在安装完 database 软件之后，dbca 最后建库时，弹出 ORA-12547 报错。

# solution

数据库依赖包未装全，缺少 baio 包，安装完后报错消除
