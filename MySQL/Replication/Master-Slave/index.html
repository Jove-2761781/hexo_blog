# Master-Slave

  1. 主库将数据变更（DML/DCL/DDL)，记录到二进制日志（Binary Log）中；

  2. 备库 IO 线程读取主库的日志，复制到自己的中继日志（Relay Log）中；

  3. 备库 SQL 线程读取中继日志中的事务，将其重放到备库。

# Procedure

  1. 主库开启二进制日志，设置 server-id

  2. 主库授予复制权限

  3. 主库备份数据
