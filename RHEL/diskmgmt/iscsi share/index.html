# iSCSI Share 磁盘共享管理

# 目录介绍

  • /backstores/block 存放本地共享设备的位置

  • /iscsi/<target>/tpg1/luns 存放加入到 target 的共享资源设备

  • /iscsi/<target>/tpg1/acls 用于存放能够访问共享存储资源的客户端名称

  • /iscsi/<target>/tpg1/portals 用于指定客户端可被访问的监听 IP 地址及端口

# iSCSI 服务端配置

  1. 安装 iSCSI，启动服务

  2. 创建虚拟设备

  3. 定义后端存储

  4. 创建 target iqn 对象

  5. 绑定后端存储

  6. 设置访问控制

  7. 绑定服务监听的地址，重启服务

# iSCSI 客户端配置

  1. 安装客户端软件，设置开机自启

  2. 将服务端访问控制表中的共享名称加入到客户端配置文件

  3. 重启客户端

  4. 扫描 iSCSI 共享存储设备

  5. 登录 iSCSI 共享服务端

  6. 查看已挂载的存储设备

  7. 删除共享服务端
