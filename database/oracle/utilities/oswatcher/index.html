# OSWatcher

# 安装

将下载好的安装包（文件名为 oswbb***.tar）上传至每个数据库节点，并解压

# 配置

  1. （可选）追踪 RAC 私有网络

    将 Exampleprivate.net 拷贝至 private.net，编辑 private.net

    Linux 平台上的 private.net 示例条目：

  2. 配置 oswbb 后台运行，每间隔 2 秒采集一次数据，采集数据保存 72 小时

  3. 编辑系统 /etc/rc.local 文件，配置 oswbb 任务开机自启动

  4. 停止 oswbb

  5. java 生成性能图表

# 卸载

删除 oswbb 目录
