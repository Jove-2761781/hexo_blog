首页 Database MySQL

通过 LVM 快照备份 MySQL 数据库

  1. 数据库设置全局读锁
    flush tables with read lock;
  2. 执行 LVM 快照
    lvcreate -L 500M -s -n mysql-snap /dev/datavg/mysql-snap
    mysql -uroot -p'Jove@123' -e 'show master status' > /backup/ date +%F _position.log
  3. 数据库释放全局读锁（关闭全局锁会话时，锁将自动失效）
    unlock tables;
  4. 挂载快照卷（只读）
  5. 从快照卷中复制数据（cp/tar）
  6. 卸载并删除快照卷

Example

backup lvm snapshot
